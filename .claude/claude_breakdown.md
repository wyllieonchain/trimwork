- Co-founded team (2 founders + 2 AI agents on the team)
- Focused on tax prep, client communication, data entry, and admin work that juniors typically handle

## AI Junior Accountant Capabilities
Shown on site as a "What they do" grid:

- **Tax return prep** — gathers source docs, populates workpapers, drafts returns
- **Client follow-ups** — chases missing documents, sends reminders
- **Data entry & reconciliation** — pulls from bank feeds, receipts, spreadsheets
- **Deadline tracking** — monitors filing deadlines, flags what needs attention
- **Workpaper organization** — sorts, labels, files docs for audit readiness
- **Client onboarding** — collects engagement letters, W-9s, prior-year returns
